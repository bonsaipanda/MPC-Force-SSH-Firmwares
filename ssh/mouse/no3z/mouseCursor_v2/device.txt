#Standard Mouse Buttons                                                                                                                                                                                            
#-------------                                                                                                                                            
#  2. BTN_RIGHT - Right mouse button (secondary click)                                                                                                                                        
#    - Default: Sends touch events                                                                                                                                                            
#    - Typically context menu button                                                                                                                                                          
#  3. BTN_MIDDLE - Middle mouse button (wheel click)                                                                                                                                          
#    - Default: Sends touch events                                                                                                                                                            
#    - Usually the clickable scroll wheel                                                                                                                                                     
#                                                                                                                                                                                             
#  Extra/Side Buttons (Gaming Mice)                                                                                                                                                           
#                                                                                                                                                                                             
#  4. BTN_SIDE - Side button (usually thumb button 1)                                                                                                                                         
#    - Also called "Mouse 4" or "Back button" on some mice                                                                                                                                    
#    - Common on gaming/ergonomic mice                                                                                                                                                        
#  5. BTN_EXTRA - Extra button (usually thumb button 2)                                                                                                                                       
#    - Also called "Mouse 5" or "Forward button" on some mice                                                                                                                                 
#    - Common on gaming/ergonomic mice                                                                                                                                                        
#  6. BTN_FORWARD - Forward navigation button                                                                                                                                                 
#    - Browser forward button                                                                                                                                                                 
#    - May overlap with BTN_EXTRA on some mice                                                                                                                                                
#  7. BTN_BACK - Back navigation button                                                                                                                                                       
#    - Browser back button                                                                                                                                                                    
#    - May overlap with BTN_SIDE on some mice                                                                                                                                                 
#  8. BTN_TASK - Task button                                                                                                                                                                  
#    - Special button on some mice                                                                                                                                                            
#    - Less common, depends on mouse model
